Google Calendar Integration

Connect your Google Calendar to enable automatic booking and reminders

Setup Instructions

1Upload your Google Service Account JSON

You'll need to upload your service account JSON file to your server.

// Example of service account JSON structure
{
  "type": "service_account",
  "project_id": "your-project-id",
  "private_key_id": "your-private-key-id",
  "private_key": "-----BEGIN PRIVATE KEY-----\\n...\\n-----END PRIVATE KEY-----\\n",
  "client_email": "your-service-account@your-project.iam.gserviceaccount.com",
  "client_id": "your-client-id",
  "auth_uri": "https://accounts.google.com/o/oauth2/auth",
  "token_uri": "https://oauth2.googleapis.com/token",
  "auth_provider_x509_cert_url": "https://www.googleapis.com/oauth2/v1/certs",
  "client_x509_cert_url": "https://www.googleapis.com/robot/v1/metadata/x509/..."
}

2Share your calendar with the service account

In Google Calendar, share your calendar with your service account email address with "Make changes to events" permissions.

Your service account email: your-service-account@your-project.iam.gserviceaccount.com

3Server-side integration code

Here's a basic example of how to use the Google Calendar API with your service account:

const { GoogleAuth } = require('google-auth-library');
const { google } = require('googleapis');

// Load the service account key JSON file
const auth = new GoogleAuth({
  keyFile: 'path-to-your-service-account-key.json',
  scopes: ['https://www.googleapis.com/auth/calendar'],
});

// Create calendar client
const calendar = google.calendar({ version: 'v3', auth });

// Example: Create a new event
async function createEvent(eventDetails) {
  try {
    const response = await calendar.events.insert({
      calendarId: 'primary', // Use 'primary' for the primary calendar
      resource: eventDetails,
    });
    return response.data;
  } catch (error) {
    console.error('Error creating calendar event:', error);
    throw error;
  }
}

4Test your integration

Once you've set up the server-side code, test if the integration is working properly.
